# 💳 sbp-qr

Красивая статичная страница для получения переводов по СБП на номер телефона.

- Приятный дизайн, оптимизирован для мобильных устройств
- Красивый QR-код
- Кнопка «Скопировать номер» + подробная инструкция
- Диплинки в приложения банков (Т-Банк, Сбер, Альфа, ВТБ и др.)
- Всё работает полностью статически на GitHub Pages — без хостинга и бэкенда

## Как пользоваться

1. Откройте: https://6w2xjystnb-bot.github.io/sbp-qr/
2. Введите фамилию, имя, отчество и номер телефона
3. Нажмите «Создать страницу»
4. Скопируйте ссылку или скачайте QR-код
5. Отправьте друзьям / выложите в профиль / био

По ссылке откроется красивая мобильная страница с:
- Вашим именем
- Кнопкой копирования номера
- Инструкцией по переводу
- Кнопками банков — при нажатии попытается открыть ваше банковское приложение с диплинком на перевод

## Своя копия

1. Сделайте Fork репозитория
2. В Settings → Pages включите GitHub Pages (Источник: Deploy from a branch → Branch: `main` / Folder: `/ (root)`)
3. Через несколько минут ваш сайт будет доступен на `https://<yourname>.github.io/sbp-qr/`

Всё бесплатно и без лишних зависимостей.

## Как работают диплинки

При нажатии на банк:
1. Номер автоматически копируется
2. Производится попытка открыть приложение банка через стандартный диплинк
3. Если приложение не открылось — появляется подсказка с инструкцией

Не использует официальные QR СБП — всё через вашу статическую страницу.
